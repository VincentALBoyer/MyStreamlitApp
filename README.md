# MyStreamlitApp

A collection of Streamlit-based business simulations and tools, including CRM, SRM, and ERP applications.

## Applications

- **CRM App**: Customer Relationship Management simulation.
- **SRM App**: Supplier Relationship Management simulation with advanced sourcing and procurement cockpit.
- **ERP App**: Enterprise Resource Planning simulation.

## Getting Started

To run the simulations, you can use the provided batch file:
```bash
launch_simulations.bat
```

Or run individual apps using Streamlit:
```bash
streamlit run SRM_App/app.py
```
